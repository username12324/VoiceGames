# LLM Usage Report

This document outlines how Artificial Intelligence and Large Language Models (LLMs) were utilized during the Sprint for Assignment 4.

## Tools Used
- **Antigravity IDE (Gemini / Claude)**: Used extensively for generating boilerplate code for new features, assisting with debugging the CI pipeline (resolving GitLab runner issues and Docker limits), refactoring the Color Match game logic, and helping structure and generate the markdown reports and documentation.

## Analysis and Original Team Effort
While AI tools assisted in the development process, the core logic, game design decisions, UX evaluations, and quality requirement definitions were the result of original team effort. The team actively reviewed, modified, and validated all AI-generated code and text to ensure it met our specific project standards and customer requirements. Specifically, the team led the customer interview, collected manual feedback, tested the applications (QA), and dictated the exact structure and content required in the documentation to the AI assistant.
