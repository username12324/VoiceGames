# Roadmap

## Current Direction
Evolving the MVP into a stable, testable product (v0.2.0) with refined voice interaction mechanics ("select-then-speak") and dynamic educational content progression.

## Milestone Roadmap

| Milestone | Timeline | Key Deliverables |
|---|---|---|
| **MVP v1 - Sprint 1** | Sprint 1 | Core voice recognition (Web Speech API), basic UI, 2 initial games (Coloring Game, Spell Caster), and GitLab Pages deployment without backend. |
| **MVP v2 - Sprint 2 (Current)** | Sprint 2 | • **Product:** Refactor voice mechanics to reduce user friction, integrate dynamic word lists, and update UI assets (pixel-art for "Heroes").<br>• **Quality & Automation:** Establish ISO/IEC 25010 based Quality Requirements, implement automated Quality Requirement Tests (QRTs) in CI, enforce a strict Definition of Done, and achieve 30% coverage on critical modules. These quality gates will be maintained in all future sprints. |
| **MVP v3 - Sprint 3 (Next)** | Sprint 3 | • Finalize the remaining mini-game prototypes to reach the 10-game goal.<br>• Introduce advanced difficulty scaling for word lists based on player progression. |
| **MVP v4 (Future)** | Future | • Parent progress dashboard (Local Storage), visual hints, and more complex game mechanics.<br>• Gamification features (stars/badges), advanced phrase processing, and additional games. |
